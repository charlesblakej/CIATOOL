Data Sources & Notes
  • Boundaries: U.S. Census Bureau TIGERweb REST services (Block Groups, Tracts, Blocks)
  • Demographics: American Community Survey (ACS) 5-year estimates via Census Data API

  • Base Map: OpenStreetMap contributors

This is a demonstration prototype. Real production version should query live TIGERweb and Census APIs (requires backend proxy for CORS). Data shown is illustrative/sample for demo purposes. Always verify with official sources for decision-making.
